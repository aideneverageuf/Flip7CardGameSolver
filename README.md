# Flip 7 Tracker

A browser-based score tracker for the card game **Flip 7**.
Also has a console-based C++ version.

## Features

- Track 2–18 players
- Automatically manages:
  - Round scoring
  - Dealer rotation
  - Deck composition
  - Bust probability
  - Flip Three
  - Freeze
  - Second Chance
- Undo the most recent action
- Works entirely in your browser
- No installation required

## Play

Visit:

[https://aideneverageuf.github.io/Flip7CardGameSolver/]

## Running Locally

Simply download the repository and open:

```
index.html
```

in any modern web browser.
